# Frontend

## Micro Frontend

Micro Frontend 是一種軟體架構模式，旨在將前端應用程式拆分成小型、獨立的模組。這些模組能夠獨立開發、測試和部署，並可在組合時共同運作，形成功能完整的前端應用。透過這種方式，可以避免單一龐大前端程式碼庫帶來的管理和維護上的困難，同時使不同團隊可以專注於各自的功能模組，增加開發效率並提升應用的可擴展性和可維護性。
